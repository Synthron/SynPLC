\chapter{Module V1}
\section{CPU}
Die CPU ist das Herz der SPS. Sie basiert auf einem STM32F1-Controller und beinhaltet folgende Funktionen:

\begin{itemize}
    \item RS485 oder CAN Businterface
    \item USB-Schnittstelle für seriellen Monitor
    \item ESP01-Modul für Webinterface
    \item SD-Kartenslot für Strukturierten Text
    \item DC/DC-Converter als Stromversorgung
\end{itemize}

Der DCDC-Converter bietet einen ESD-geschützten 24V-Eingang und liefert 5V bei 10W Ausgangsleistung. Diese Spannung liegt am Busstecker an und versorgt die anderen Module.

\section{DO}
Das DO (Digital-Output) stellt 8 24V-Ausgänge mit einer Belastbarkeit von jeweils 0,5A zur Verfügung. Die 24V-Spannung wird extern angelegt. 
Gleichzeitig werden die Ausgänge rückgelesen, sodass der Status angezeigt werden kann.

Die LEDs auf der Frontplatte zeigen an, ob ein Kanal High oder Low ist. Dadurch wird ein optisches Feedback gegeben. 

Das DO hat einen Schutzmechanismus gegen Überlastung der Ausgänge. Wenn der Controller bemerkt, dass ein Ausgang kurzgeschlossen ist, wird dieser deaktiviert. Nun werden alle 100ms der Ausgang wieder aktiviert und nachgeschaut, ob der Kurzschluss behoben wurde. Wenn ja, bleibt der Ausgang aktiviert, wenn nicht, wird das "Polling" fortgesetzt, bis die CPU den Ausgang deaktiviert. 

Während einem aktiven Fehler ist das jeweilige Bit im Fehlerspeicher gesetzt. Beim Zugriff der CPU auf das Modul wird dieser Speicher einmal pro Zyklus ausgelesen. 

\section{DI}
Das DI bietet 8 Eingänge mit ESD-Schutz und zwei galvanisch getrennten Potentialen. Ein High-Pegel wird ab einer Eingangsspannung von ca. 7,5V erkannt. Die Eingänge sind bis 33V spannungsfest und verbrauchen einen Maximalstrom von 2mA. 

Auf der Frontplatte ist eine LED je Kanal, welche den Status anzeigt. 

\section{AO}
Das AO stellt vier analoge Ausgänge zur Verfügung, die jeweils entweder 0 bis 10V Konstantspannung oder 0 bis 20mA Konstantstrom liefern. Die Umschaltung erfolgt dabei über Relais. 

\section{AI}
Das AI stellt vier analoge Eingänge bereit, die jeweils entweder 0 bis 10V Konstantspannung oder 0 bis 20mA Konstantstrom messen können. Die Umschaltung erfolgt über MOSFETs. 

\section{XT}
Das XT stellt vier Triac-Ausgänge auf zwei Spannungsdomänen bereit. Jede Domäne verfügt über eine eigene Nulldurchgangserkennung. Die Kanäle können max 230V und 1A regeln.
