\chapter{Bus-Systeme}
Die CPU unterstützt nativ zwei Bus-Physiken:
\begin{itemize}
    \item RS485
    \item CAN
\end{itemize}

Für die IO-Module V1 wird nur RS485 unterstützt. Dazu wird auf der CPU die Steckbrücke für den Backplane entsprechend gesetzt.
